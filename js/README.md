# digitaldias JavaScript Directory

This directory contains all JavaScript files for the digitaldias website.

## Current files:
- main.js - Main JavaScript file with interactive features and animations

## JavaScript Features:
The main.js file includes:
- Page initialization and event listeners
- Interactive logo and button animations
- Parallax mouse tracking effects
- Floating particle animations
- Modal dialogs for feature details and coming soon messages
- Easter egg functionality
- Background color animations
- Analytics tracking (placeholder)

## Recommended JS structure for larger projects:
- main.js - Main entry point (current)
- utils.js - Utility functions
- components/ - Individual component JavaScript files
- animations.js - Animation and visual effects
- analytics.js - Analytics and tracking
- api.js - API calls and data handling

## Usage:
The main JavaScript file is loaded at the end of the HTML body:
```html
<script src="js/main.js"></script>
```
