# digitaldias CSS Directory

This directory contains all stylesheet files for the digitaldias website.

## Current files:
- styles.css - Main stylesheet with all base styles, animations, and responsive design

## Recommended CSS structure for larger projects:
- styles.css - Main stylesheet (current)
- reset.css - CSS reset/normalize styles
- variables.css - CSS custom properties and variables
- components.css - Reusable component styles
- utilities.css - Utility classes
- responsive.css - Media queries and responsive styles
- animations.css - Keyframe animations and transitions

## Usage:
The main stylesheet is linked in the HTML head:
```html
<link rel="stylesheet" href="css/styles.css">
```

## CSS Architecture:
The current styles.css follows a organized structure:
1. Reset and base styles
2. Layout and container styles
3. Component styles (logo, features, etc.)
4. Animations and keyframes
5. Responsive design (media queries)
