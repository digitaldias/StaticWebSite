# digitaldias Icons Directory

This directory contains icons and favicon files for the digitaldias website.

## Files to add:
- favicon.ico - Main website favicon
- favicon-16x16.png - 16x16 favicon
- favicon-32x32.png - 32x32 favicon
- apple-touch-icon.png - Apple touch icon
- android-chrome-192x192.png - Android chrome icon (192x192)
- android-chrome-512x512.png - Android chrome icon (512x512)

## Usage:
The favicon.ico file should be referenced in the HTML head section:
```html
<link rel="icon" type="image/x-icon" href="icons/favicon.ico">
```

For a complete favicon setup, you can generate the required files using online favicon generators and place them in this directory.
