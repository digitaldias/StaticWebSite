# digitaldias Images Directory

This directory contains all image assets for the digitaldias website.

## Recommended image structure:
- backgrounds/ - Background images and textures
- logos/ - Company logos in various formats
- features/ - Feature illustrations and icons
- team/ - Team member photos
- portfolio/ - Portfolio and project images
- banners/ - Marketing banners and promotional images

## Image optimization guidelines:
- Use WebP format for modern browsers with JPEG/PNG fallbacks
- Optimize images for web (compress without losing quality)
- Use responsive images with appropriate sizes
- Include alt text for accessibility

## Example usage:
```html
<img src="images/logos/digital-dias-logo.png" alt="digitaldias Logo" width="200" height="80">
```
